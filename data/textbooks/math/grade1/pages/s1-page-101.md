# مَشْروعُ الْوَحْدَةِ: الْفُصولُ الْأَرْبَعَةُ

أَسْـتَعِدُّ أَنا وَمَجْموعَتي لِتَنْفيذِ مَشْـروعِنا الْخاصِّ بِالْفَصْلِ الْمُفَضَّلِ، بِناءً عَلى ما سَـسْتَعَلَّمُهُ في هذِهِ الْوَحْدَةِ.

## الْمَوادُ وَالْأَدَواتُ

- أَلْوانٌ
- وَرَقَةٌ كَرْتونِيَّةٌ كَبيرَةٌ
- أَفْلامُ تَلْوينٍ

## خُطُواتُ تَنْفيذِ الْمَشْروعِ:

**1** أَسْأَلُ 20 زَميلًا/ زَميلَةً عَنِ الْفَصْلِ الَّذي يُفَضِّلُهُ كُلُّ مِنْهُمْ/ مِنْهُنَّ (الرَّبيـعُ، الصَّيْفُ، الْخَريفُ، الشِّتاءُ).

**2** أُنَظِّمُ الْبَياناتِ في الْجَدْوَلِ الْمُجاوِرِ.

| الْفَصْلُ | الْعَدَدُ |
|---|---|
| الرَّبيعُ | |
| الصَّيْفُ | |
| الْخَريفُ | |
| الشِّتاءُ | |

**3** أُمَثِّلُ هذِهِ الْبَياناتِ بِالصُّوَرِ.

**4** أُحَدِّدُ الْفَصْلَ الَّـذي يُفَضِّلُهُ أَكْثَـرُ عَـدَدٍ مِنَ الزُّمَلاءِ/ الزَّميلاتِ، وَالْفَصْلَ الْأَقَلَّ تَفْضيلًا لَدَيْهِمْ/ لَدَيْهِنَّ.

**5** أَعْرِضُ عَلى الْوَرَقَةِ الْكَرْتونِيَّةِ الْكَبيرَةِ جَدْوَلَ الْبَياناتِ، وَالتَّمْثيلَ بِالصُّوَرِ، وَالتَّفْسيراتِ الَّتي تَوَصَّلْتُ إِلَيْها ثُمَّ أُعَلِّقُها عَلى لَوْحَةِ الصَّفِّ.
