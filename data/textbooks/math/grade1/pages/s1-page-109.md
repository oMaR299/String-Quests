# أَتَحَقَّقُ مِنْ فَهْمي ✓

سَأَلَ مُرادٌ طَلَبَةَ صَفِّهِ عَنِ الْفاكِهَةِ الْمُفَضَّلَةِ لَدَيْهِمْ، فَكانَتْ إِجاباتُهُمْ كَما هُوَ مُبَيَّنٌ في الصُّنْدوقِ الْآتي:

<!-- SVG_TODO: A box containing scattered fruit images - multiple apples (التُّفّاحُ), oranges (الْبُرْتُقالُ), and bananas (الْمَوْزُ) representing survey responses. Students need to count each type. -->

> **1** أُمَثِّلُ الْبَياناتِ بِالصُّوَرِ.
>
> | الْفاكِهَةُ الْمُفَضَّلَةُ | |
> |---|---|
> | التُّفّاحُ 🍎 | |
> | الْبُرْتُقالُ 🍊 | |
> | الْمَوْزُ 🍌 | |
>
> الْمِفْتاحُ: كُلُّ 😊 تُمَثِّلُ طالِبًا.

> **2** أُمَثِّلُ بِالصُّوَرِ الْبَياناتِ الْآتِيَةَ الَّتي تُشيرُ إِلى هِوايَاتِ بَعْضِ الْأَشْخاصِ الْمُفَضَّلَةِ:
>
> | الْهِوايَةُ الْمُفَضَّلَةُ | | | الْهِوايَةُ الْمُفَضَّلَةُ | |
> |---|---|---|---|---|
> | الْمُطالَعَةُ 📖 | 5 | | الْمُطالَعَةُ 📖 | |
> | الرَّسْمُ 🎨 | 3 | | الرَّسْمُ 🎨 | |
> | التَّصْويرُ 📷 | 8 | | التَّصْويرُ 📷 | |
>
> الْمِفْتاحُ: كُلُّ 😊 تُمَثِّلُ شَخْصًا.

## أَحُلُّ الْمَسْأَلَةَ

> **3** أُبَرِّرُ: إِذا سَأَلْنا 5 أَشْخاصٍ آخَرينَ عَنِ الْهِوايَةِ الْمُفَضَّلَةِ لَدَيْهِمْ، فَهَلْ سَيَتَغَيَّرُ التَّمْثيلُ السّابِقُ؟
>
> أُبَرِّرُ إِجابَتي.

**نَشاطٌ مَنْزِلِيٌّ:** أَطْلُبُ إِلى طِفْلي/ طِفْلَتي تَمْثيلَ وَجْبَةِ الطَّعامِ الْمُفَضَّلَةِ لِأَفْرادِ الْأُسْرَةِ بِاسْتِعْمالِ الصُّوَرِ.
